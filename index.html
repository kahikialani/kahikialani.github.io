CAL FIRE Analytics Pipeline

Built a Python + SQL pipeline processing 3,200+ CAL FIRE incidents, including data cleaning, validation, and SQLite storage. Created Tableau dashboards and statistical analyses revealing seasonal wildfire patterns, and engineered a workflow supporting predictive modeling for fire risk assessment.

Forecasting Spotify Interest

Analyzed 13 years of Google Trends data in R to model and forecast Spotify search popularity. Built a SARIMA(2,1,1)x(1,0,0) model achieving 89% forecast accuracy, complemented with GARCH modeling to capture volatility patterns. Identified cyclical peaks every 12/15 months and visualized long-term trends, providing actionable insights for strategic planning in music streaming analytics.

Quantifying Rock Climbing Ability with ML

Built multiple predictive models in R to estimate climbing performance from user metrics, including Random Forests, SVM, k-NN, and LDA, achieving 75% test accuracy. Cleaned and preprocessed public survey data, implemented 10-fold cross-validation, and developed an interactive interface to visualize how strength metrics influence climbing grades.
